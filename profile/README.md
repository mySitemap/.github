## 🚀 Overview

**mySitemap** is a macOS application designed for web designers, developers, and project managers who need to quickly map out, plan, and visualize website structure. Instead of relying on static sketches or manual charts, mySitemap offers a dynamic, drag-and-drop interface for creating interactive sitemaps that make sense at a glance.  

You can visually organize web pages into hierarchies, add metadata like titles, URLs, descriptions, and define navigation flow. This makes it easier to communicate ideas with clients, team members, or stakeholders. The app is especially useful in the early stages of a project, where clear planning prevents costly restructuring later.  

With mySitemap, exporting is seamless: create sharable diagrams as **PDF, PNG, or HTML interactive sitemap files**. These exports are ideal for documentation, client presentations, or internal design specifications. Collaboration is enhanced through clear visualization and flexibility — changes can be made on the fly and reflected in real time.  

The app is lightweight yet powerful, with focus on usability. It integrates well into web development workflows, reduces communication friction, and helps align teams on project scope and navigation flow. Whether you’re a freelancer building small business sites or an agency managing enterprise-level platforms, mySitemap saves time and increases clarity.
